- *Supply and Demand Model*
  + *Demand -* How much people want it! /(The greater the demand, the higher the price)/
  + *Supply -* How much is available /(The greater the supply, the lower the price)/
  + *Equilibrium point -* The point in which both the consumer and the provider are happy to purchase and provides the goods /(This is where the "demand" curve intersect with the "supply" curve)/
